## Darcy -- A system for generating ESLint Files

### Installation

1. `npm install`

### Running
1. `npm run watch`

### Using
Go to [http://localhost:3000/ui/add.html](http://localhost:3000/ui/index.html)

### Infiltrating

Try to add an ESLint rule with the following _value_:

